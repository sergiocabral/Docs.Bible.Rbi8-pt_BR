[[v22-7-0]]
=== Cântico de Salomão, 7
// Rbi8 1986, Rbi8 - Tradução do Novo Mundo das Escrituras Sagradas com Referências

[[v22-7-1]]

^<<v22-7-1,1>>^ “Quão belos se tornaram os teus passos nas [tuas] sandálias, ó filha disposta! As curvaturas das tuas coxas são como ornamentos, trabalho das mãos dum artesão.

[[v22-7-2]]

^<<v22-7-2,2>>^ Teu umbigo é uma taça redonda. Não [lhe] falte o vinho misturado. Teu ventre é um monte de trigo, cercado de lírios.

[[v22-7-3]]

^<<v22-7-3,3>>^ Teus dois peitos são como duas crias, gêmeas duma fêmea de gazela.

[[v22-7-4]]

^<<v22-7-4,4>>^ Teu pescoço é como torre de marfim. Teus olhos são como as lagoas de Hésbon, junto ao portão de Bate-Rabim. Teu nariz é como a torre do Líbano, que olha para Damasco.

[[v22-7-5]]

^<<v22-7-5,5>>^ Tua cabeça sobre ti é como o Carmelo, e as madeixas de tua cabeça são como lã tingida de roxo. O rei é mantido preso pelas ondulações.

[[v22-7-6]]

^<<v22-7-6,6>>^ Quão bela és e quão agradável és, ó amada,^&#42;1^ entre delícias!

[NOTE]
====
. Lit.: “ó amor”.
====

[[v22-7-7]]

^<<v22-7-7,7>>^ Esta estatura tua se assemelha a uma palmeira, e teus peitos, a cachos de tâmaras.

[[v22-7-8]]

^<<v22-7-8,8>>^ Eu disse: ‘Subirei na palmeira, para apoderar-me dos seus ramos de tâmaras.’ E, por favor, tornem-se os teus peitos como os cachos da videira, e a fragrância de teu nariz, como maçãs,

[[v22-7-9]]

^<<v22-7-9,9>>^ e teu paladar, como o melhor vinho que escorre suavemente para meu amor, deslizando sobre os lábios dos adormecidos.”

[[v22-7-10]]

^<<v22-7-10,10>>^ “Eu sou do meu querido, e seu desejo ardente é para comigo.

[[v22-7-11]]

^<<v22-7-11,11>>^ Vem deveras, ó meu querido, saiamos ao campo; pousemos entre as plantas de hena.^&#42;1^

[NOTE]
====
. “As aldeias”, LXXSyVg. Mas, veja 1:14; 4:13.
====

[[v22-7-12]]

^<<v22-7-12,12>>^ Levantemo-nos deveras cedo e vamos aos vinhedos, para que vejamos se a videira floresceu, se a flor se abriu, se as romãzeiras brotaram. Ali te darei as minhas expressões de afeto.

[[v22-7-13]]

^<<v22-7-13,13>>^ As próprias mandrágoras deram a [sua] fragrância, e junto às nossas entradas há toda sorte de frutas seletas. As novas bem como as antigas, ó meu querido, eu entesourei para ti.
